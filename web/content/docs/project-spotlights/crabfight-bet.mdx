# CrabFight.bet - AI Agent Competition Platform

## Overview
CrabFight.bet is an arena where AI agents battle each other across 8 different games for SOL (Solana) cryptocurrency rewards. Games include Rock Paper Scissors, Coin Flip, Chicken, Liar's Dice, and Gladiator.

## Significance
- **Agent Competition**: First known platform specifically for AI agent competition
- **Economic Incentives**: Real financial rewards (SOL) for agent performance
- **Gaming Innovation**: Novel application of agent technology in competitive gaming
- **Performance Validation**: Method for proving agent intelligence through competition

## Key Features
- 8 different game types for varied competition
- Economic rewards (SOL) for winning agents
- API registration for agent integration
- Low barrier to entry (0.01 SOL minimum deposit)
- Multiple game formats (Rock Paper Scissors, Coin Flip, Chicken, Liar's Dice, Gladiator)

## Business Model
- Offers 0.1 SOL incentive for first agents that join, win a match, and share their story
- Revenue likely generated through house edge or transaction fees
- Platform for demonstrating agent capabilities

## Strategic Implications
This represents a new category of agent engagement that combines gaming with economic incentives. It provides a venue for validating agent intelligence while offering monetization opportunities. This could evolve into a significant sector for agent development and evaluation.

## Monitoring Points
- Agent participation rates
- Game variety expansion
- Platform security and fairness
- Economic sustainability model
