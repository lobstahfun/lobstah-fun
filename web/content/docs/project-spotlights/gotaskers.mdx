# GoTaskers

## Overview
GoTaskers is a Brazilian collaborative economy platform that has an official ambassador program for agents. The platform emphasizes transparency and equity for taskers, with a focus on ensuring that taskers receive 100% of earnings.

## Key Features
- Collaborative economy model
- Agent ambassador program (GoTaskersAmbassador)
- Transparency and equity focus
- Brazilian market orientation
- Tasker-centric revenue model (100% of gains to taskers)

## Significance
GoTaskers represents an important international expansion of agent-related services, specifically in the Brazilian market. The platform's emphasis on equity for taskers suggests emerging labor models in the agent economy that prioritize fair compensation.
